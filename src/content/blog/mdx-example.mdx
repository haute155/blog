MDX를 사용하면 Markdown 문법 그대로 쓰면서 React 컴포넌트를 자유롭게 삽입할 수 있습니다.

## Image 컴포넌트

`src`, `alt`는 필수이고 `caption`은 선택입니다.

<Image
  src="https://images.unsplash.com/photo-1555066931-4365d14bab8c?w=1200&q=80"
  alt="코드가 보이는 모니터"
  caption="사진 출처: Unsplash"
/>

## Callout 컴포넌트

`type`은 `info` (기본값) / `tip` / `warning` 세 가지입니다.

<Callout type="info">
  이 글은 MDX 기본 설정 예시입니다. 컴포넌트를 직접 import해서 쓰거나,
  `[slug].astro`에서 전역으로 주입된 컴포넌트를 그대로 사용할 수 있습니다.
</Callout>

<Callout type="tip" title="알아두면 좋아요">
  전역 주입된 컴포넌트(`Image`, `Callout`)는 import 없이 바로 쓸 수 있습니다.
</Callout>

<Callout type="warning">
  외부 이미지를 사용할 때는 Astro의 `image()` 스키마를 활용하면 최적화도 함께
  처리할 수 있습니다.
</Callout>

## 일반 Markdown도 그대로

MDX 안에서 기존 Markdown 문법은 전부 동작합니다.

- 리스트
- **굵게**, _기울임_, `인라인 코드`

```tsx
// 코드 블록도 그대로
export default function Hello() {
  return <p>Hello MDX!</p>;
}
```
